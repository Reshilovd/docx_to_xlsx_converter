# DOCX to XLSX Converter (React)

React-приложение для конвертации `.docx` в `.xlsx` прямо в браузере.

## Как работает

- Для чтения `.docx` используется `mammoth` (конвертация в HTML).
- Можно выбрать несколько `.docx` файлов и режим вывода:
  - по колонкам (старый формат),
  - в одну ячейку как HTML со `span`.
- Для режима `span` указывается CSS-класс для каждого файла (например `ru`, `kg`).
- Строки выравниваются по кодам вопросов (`A1.`, `B2.`, `98.`).
- Для записи Excel используется `exceljs` с сохранением базового форматирования текста (жирный, курсив, подчеркивание, переносы строк).
- Никакой backend и LibreOffice не требуются.

## Запуск

```powershell
npm install
npm start
```

Приложение откроется на `http://localhost:3000`.
