# DOCX to XLSX Converter

Инструмент для сборки многоязычных анкет из DOCX в XLSX.

Основной сценарий работает в браузере (React + ExcelJS), без обязательного backend.

## Основной поток (frontend)

1. Загрузите несколько `.docx` файлов (одна структура, разные языки).
2. Скачайте промежуточный `*-columns.xlsx`.
3. Проверьте и при необходимости поправьте строки вручную.
4. Загрузите проверенный `.xlsx` на шаге 2 и склейте в итоговый `*-spans.xlsx`.

## Что сохраняется

- Выравнивание строк по кодам вопросов (`A1.`, `B2.`, `98.`).
- Защита от сдвига вариантов ответа: служебные строки вида `(...)` и заголовки `Блок/Block` не сбивают индексацию ответов.
- Форматирование текста сохраняется программно через `richText` Excel (жирный, курсив, подчеркивание, перечеркивание, над/подстрочный индекс, цвет, размер, шрифт).
- Для каждой языковой части добавляются текстовые обертки `<span class="...">...</span>` с настраиваемыми классами.

## Локальный запуск frontend

```powershell
npm install
npm start
```

Откроется `http://localhost:3000`.

## Тесты

```powershell
npm test -- --watchAll=false
```

## Дополнительно: backend-реализации

В репозитории есть две альтернативные backend-реализации:

- `server/index.js` — Express + LibreOffice (`/api/convert`).
- `app/main.py` — FastAPI + python-docx/openpyxl (`/convert`).

Они не обязательны для основного браузерного сценария и могут использоваться как отдельные сервисы.
